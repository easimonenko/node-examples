# node-examples

## net-tcp

Example for modules [Net](https://nodejs.org/api/net.html) over TCP and [Events](https://nodejs.org/api/events.html).

NPM-packages: only standard packages.

Run server

``` bash
npm run net-tcp
```

or

``` bash
node server.js
```

Connect to server with clients:

``` bash
nc localhost 8888
```
