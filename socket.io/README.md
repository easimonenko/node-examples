# node-examples

## socket.io

Example for Socket.IO based on "Node.js in Action" (Cantelon, Harter and others).

[Socket.IO](https://socket.io/)

- NPM-package:

`socket.io`

Also need to be installed `mime`.

- run server

``` sh
npm run socket.io
```

- open in browser

``` plain
http://localhost:3000
```
